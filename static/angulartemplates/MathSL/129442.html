Question

Let \(f(x) = {x^2} - 4x + 5\).

The function can also be expressed in the form \(f(x) = {(x - h)^2} + k\).

Find the equation of the axis of symmetry of the graph of \(f\).

[2]
a.

(i)     Write down the value of \(h\).

(ii)     Find the value of \(k\).

[4]
b.

\(x = 2\) (must be an equation)     A1     N2
