METHOD 2 (finding r first)

valid approach     (M1)

eg\(\,\,\,\,\,\)\(r = \frac{6}{{x - 3}},{\text{ }}6r = x + 2,{\text{ }}(x - 3){r^2} = x + 2\)

eg\(\,\,\,\,\,\)\(\frac{6}{r} + 3 = 6r - 2,{\text{ }}6 + 3r = 6{r^2} - 2r,{\text{ }}6{r^2} - 5r - 6 = 0\)
