Question

Let \(f (x) = 2x^2 + x - 6\)

Find \(f'(x)\).

[3]
a.

Find the value of \(f'( - 3)\).

[1]
b.

Find the value of \(x\) for which \(f'(x) = 0\).

[2]
c.

Markscheme

\(f'(x) = 4x + 1\)     (A1)(A1)(A1)     (C3)


Note: Award (A1) for each term differentiated correctly.

Award at most (A1)(A1)(A0) if any extra terms seen.

 

[3 marks]

a.

\(f'( - 3) =  - 11\)     (A1)(ft)     (C1)

[1 mark]

b.

\(4x + 1 = 0\)     (M1)

\(x = - \frac{{1}}{{4}}\)     (A1)(ft)     (C2)

[2 marks]

c.
