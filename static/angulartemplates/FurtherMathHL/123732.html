Question

The function \(f:\mathbb{R} \to \mathbb{R}\) is defined by \(f:x \to \left\{ {\begin{array}{*{20}{c}} { - 3x + 1}&{{\text{for }}x < 0} \\ 1&{{\text{for }}x = 0} \\ {2{x^2} - 3x + 1}&{{\text{for }}x > 0} \end{array}} \right.\).

By considering limits prove that \(f\) is

continuous at \(x = 0\);

[4]
a.

differentiable at \(x = 0\).

[5]
b.

Markscheme

consider \(\mathop {\lim }\limits_{h \to 0 + } f(0 + h) = \mathop {\lim }\limits_{h \to 0 + } (2{h^2} - 3h + 1)\)     M1

\( = 1 = f(0)\)    A1

\(\mathop {\lim }\limits_{h \to 0 - } f(0 + h) = \mathop {\lim }\limits_{h \to 0 - } ( - 3h + 1)\)    M1

\( = 1 = f(0)\)    A1

hence \(f\) is continuous at \(x = 0\)     AG

 

Note:     The \( = f(0)\) needs only to be seen once.
