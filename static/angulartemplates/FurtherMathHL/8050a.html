then,

\(a + e - ae = a\)     A1

\(e(1 - a) = 0\)

\(e = 0\)     A1

 

(iii)     let \({a^{ - 1}}\) be the inverse of \(a\), so that \(a * {a^{ - 1}} = 0\)     (M1)

\(a + {a^{ - 1}} - a{a^{ - 1}} = 0\)     A1

\({a^{ - 1}} = \frac{a}{{a - 1}}\)     A1

this gives an inverse for all elements except 1 which has no inverse     R1

 

(iv)     \(2x - {x^2} = 1\)     M1

\({(x - 1)^2} = 0\)     (A1)

\(x = 1\)     A1
