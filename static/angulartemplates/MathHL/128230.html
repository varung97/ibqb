Question

Using the information on the graph,

state the value of \(a\) and the value of \(c\);

[2]
a.

find the value of \(b\).

[2]
b.

Markscheme

\(a = 1\)    A1

\(c = 3\)    A1

[2 marks]

a.

use the coordinates of \((1,{\text{ }}0)\) on the graph     M1

\(f(1) = 0 \Rightarrow 1 + \frac{b}{{1 - 3}} = 0 \Rightarrow b = 2\)    A1
