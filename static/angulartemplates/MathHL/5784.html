Question

Write \(\ln ({x^2} - 1) - 2\ln (x + 1) + \ln ({x^2} + x)\) as a single logarithm, in its simplest form.

Markscheme

\(\ln ({x^2} - 1) - \ln {(x + 1)^2} + \ln x(x + 1)\)     (A1)

\( = \ln \frac{{x({x^2} - 1)(x + 1)}}{{{{(x + 1)}^2}}}\)     (M1)A1

\( = \ln \frac{{x(x + 1)(x - 1)(x + 1)}}{{{{(x + 1)}^2}}}\)     (A1)

\( = \ln x(x - 1)\,\,\,\,\,\left( { = \ln ({x^2} - x)} \right)\)     A1
