\({\text{P}}({S_1} + {S_2} + {S_3} + {S_4} + {S_5} < 20.5)\)

let \({S_1} + {S_2} + {S_3} + {S_4} + {S_5} = A\)     (M1)

\({\text{E}}(A) = 5{\text{E}}(S)\)

= 21     A1

\({\text{Var}}(A) = 5{\text{Var}}(S)\)

= 0.1125     A1

\(A \sim {\text{N(21, 0.1125}})\)

\({\text{P}}(A < 20.5) = 0.0680\)     A1

= 0.302     A1
