By first considering the elements \({2^1},{\text{ }}{2^2},{\text{ ..., }}{2^k}\) and then the elements \({2^{k + 1}},{\text{ }}{2^{k + 2}},{\text{ …,}}\) show that the order of the element 2 is 2k.
