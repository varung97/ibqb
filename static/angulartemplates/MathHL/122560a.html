\({x^2} = \frac{{1 - y}}{{1 + y}}\)

\({x^2} + {x^2}y = 1 - y\)    M1

Note:     Award M1 for attempting to make \(y\) the subject.

\(y(1 + {x^2}) = 1 - {x^2}\)    (A1)

\({f^{ - 1}}(x) = \frac{{1 - {x^2}}}{{1 + {x^2}}},{\text{ }}x \geqslant 0\)    A1A1

Note:     Award A1 only if \({f^{ - 1}}(x)\) is seen. Award A1 for the domain.

the range of \({f^{ - 1}}\) is \( - 1 < {f^{ - 1}}(x) \leqslant 1\)     A1
